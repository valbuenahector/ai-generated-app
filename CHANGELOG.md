# Changelog

All notable changes to this project will be documented in this file.

## [0.2.5] - 2026-01-06
### Changed
- Replaced F5 ADSP Hero PNG with Gemini generated AI visual on Home page.
- Adjusted Hero image alignment to `object-top` to prevent cutting off critical "AI" text in the visual.

## [0.2.4] - 2026-01-06
### Changed
- Removed Health Check button from header to prepare for Module 3 login integration.
- Replaced ADSP Simplified SVG with high-quality Hero PNG in Home page.
- Removed placeholder text from Home page security visual.
- Removed "Primary Source Material" section from Docs page for a cleaner interface.

## [0.2.2] - 2026-01-06
### Changed
- Robust global centering of the application within the browser window using a max-width wrapper and CSS reset.
- Integrated official F5 logo (`f5-logo-rgb.svg`) in the header.
- Integrated F5 ADSP Graphic (`F5-ADSP-Graphic-v2.0_Simplified.svg`) in the Home page hero.
- Refined Docs page: white headline and simplified content to Benefits and Risks only.

## [0.2.1] - 2026-01-06
### Changed
- Removed "Module 2 Baseline" from header menu.
- Improved layouts for "About" and "Docs" pages with better spacing and visual hierarchy.
- Refined Home page Hero section with white headline for better contrast.
- Ensured consistent centering and responsiveness across all pages.

## [0.2.0] - 2026-01-06
### Added
- Complete UI/UX redesign using F5 Brand Guidelines.
- Integration of CDN-hosted Tailwind CSS.
- Modern Home Page layout with Hero section, History, Comparison, and FAQ.
- Expanded content on Vibe Coding with proper citations.
- Responsive, container-friendly layout enhancements.

## [0.1.0] - 2026-01-06
### Added
- Initial AI-generated baseline for Module 2.
- Basic Flask application structure.
- Read-only routes for Home, About, and Docs.
- Minimal CSS styling.
- Dockerfile for containerized deployment.
- Basic unit tests for routes.
- Intentional (lab-safe) security weaknesses for demonstration.
- PRD, TODO, and README documentation.

This version is the product of vibe coding and serves as a starting point for the AppWorld 2026 lab.
